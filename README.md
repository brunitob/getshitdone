####Let's go#####
Checkout this repo, install dependencies, then start the process with the following:

```
	> git https://github.com/brunitob/getshitdone.git
	> cd getshitdone
	> npm install
	> npm start
```

####What Im using?#####
- React
- Redux
- Routing
- Redux Persist (refresh-proof)
